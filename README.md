# Personal Portfolio Website

This repository contains the source code for my personal portfolio website, hosted at [ztree.dev](https://ztree.dev). 

The site serves as a central hub for my resume, technical projects, and background in software development, system administration, and cloud computing.

##  Built With

This portfolio was designed to be lightweight, responsive, and easy to maintain. It is built using:

* Bun - JavaScript Runtime
* Vue.js 3 - Reactive frontend framework
* Tailwind CSS - Utility first CSS framework
* Vite - Bundler for development and production
* Cloudflare Pages - CI/CD and deployment
* Dependencies: https://github.com/Ejz9/ejz9.github.io/blob/main/package.json

## Local Development

To run this website locally for development or testing:

1. Clone the repository:
   ```bash
   git clone https://github.com/Ejz9/ejz9.github.io.git
   or
   git clone git@github.com:Ejz9/ejz9.github.io.git
   cd ejz9.github.io
    ```
2. Install the required dependencies:
   ```bash
   bun install # or npm/yarn/pnpm depending on your setup
   ```
3. Start the local development server:
   ```bash
   bun run dev
   ```
